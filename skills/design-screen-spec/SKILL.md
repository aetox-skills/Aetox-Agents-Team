# Screen Specification Standard

Load this skill when asked to design a screen or component in detail. Output all 17 sections for production-ready specs.

## 1. Purpose
What problem does this screen/component solve? Why does it exist?

## 2. User Context
Who uses it, when, why, on what device, in what environment?

## 3. Primary Task
The single thing the user must accomplish on this screen.

## 4. Primary Action
The most important action button/gesture/link.

## 5. Secondary Actions
Useful but lower-priority actions.

## 6. Information Hierarchy
Rank content from most important to least important.

## 7. Layout Structure
Sections, order, grouping, spacing, responsive breakpoints.

## 8. Components
List every component: Button, Input, Select, Table, Card, Modal, Drawer, Tabs, Toast, Badge, Tooltip, Navigation, Empty/Loading/Error state.

## 9. Interaction States
For each interactive element: default, hover, focus, active, disabled, loading, success, error, empty, offline, permission denied, conflict, retry, confirmation.

## 10. Validation Rules
Form validation, data constraints, permission checks, business rules, edge cases.

## 11. Error Recovery
How users fix mistakes: inline edit, undo, cancel, retry, contact support.

## 12. Accessibility Notes
Contrast ratio (min 4.5:1 text), keyboard tab order, focus indicators, screen reader labels/roles, touch targets (min 44px), reduced motion, color-blind safe signals.

## 13. UX Copy
Labels, CTAs, helper text, placeholder, error text, empty state, success message, confirmation text, tooltip.

## 14. Responsive Rules
Mobile (<640px), Tablet (640-1024px), Desktop (>1024px). Navigation changes, grid behavior, density, touch vs mouse targets.

## 15. Developer Handoff
Implementation notes, reusable components, CSS tokens (colors, spacing, radii), edge states, loading skeletons, empty state layout.

## 16. Risks
Design risks (unclear hierarchy), usability risks (slow flow), technical risks (API latency), content risks (vague copy).

## 17. Success Metrics
How to measure success: completion rate, time on task, error rate, drop-off, satisfaction score.
